# JP CABS — Premium Cab Service in Pondicherry

Premium cab services in Pondicherry specializing in airport transfers, local sightseeing, and outstation trips.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI
- Lucide React
- TanStack Query
- React Router DOM
- Sonner
- Zod
- React Hook Form

## Features

- **Premium Fleet**: Wide range of vehicles for all needs.
- **Airport Transfers**: Reliable pickup and drop-off.
- **Local Sightseeing**: Explore Pondicherry with local experts.
- **Outstation Trips**: Comfortable long-distance travel.
- **24/7 Service**: Available whenever you need us.

## Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
